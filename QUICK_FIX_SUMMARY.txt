## 🔐 Login Flow

```
User selects role
     ↓
Enters email + password
     ↓
Backend validates credentials
     ↓
Returns user with ROLE
     ↓
Frontend checks: role matches selected?
     ↓
✅ YES → Store token + role → Redirect to dashboard
❌ NO  → Show error "Account is not a [role]"
```
